Calculation Parameters

Select the Start Year and End Year for the year of diagnosis range.

Select Cohort Variables,Covariate Variables and the appropiate values for Cohort Values for the SEER Join Point Survivability calculation.

  • Select one of more Cohort Variables
  • Select Cohort Values
  • Select Covariate Variable if avalable

Select the number of Join Points to use between the interval selected. Selecting 3 or more Join Points may take a considerable amount of time. When long calculations are completed an e-mail will be sent containing a link to the results.

Click Calculate to execute the Join Point calculation and view the APC table.

Click Reset to reset all values and start a new SEER Join Point calculation.
